Initializing Barrage (JS & HTML & CSS) 
        
  var displayRegion = document.querySelector('.display__region');
  displayRegion.style.height = '200px';
  displayRegion.style.width = '800px';

  var structure = {
    poolNum: 5,
    channelNum: 10,
    sentDelay: 2,
    recurrence: {
      recur: true,
      delay: 7
    },
    comment: {
      height: 40,
      width: 100,
      size: 14,
      color: 'black',
      hover: true
    },
    animation: {
      ani: null,
      dir: null,
      random: false
    }
  }

  var barrage = new Barrage(displayRegion, structure)

Global Variables in Barrage
Property Default Type Description
displayRegion / Object The DOM element for displaying Barrage. Need to set height and width of the displayRegion before initializing Barrage. Note: displayRegion is not in the initial Barrage structure.
commentHeight 40 Int The number of pixels of the height for the global comment texts. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.comment.height for your Barrage.
commentWidth 100 Int The number of pixels of the width for the global comment texts. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.comment.width for your Barrage.
commentSize 14 Int The font size for the global comment texts. Can be changed after initializing Barrage. Minimum: 10px. For the initial Barrage structure, set structure.comment.size for your Barrage.
commentColor 'black' HEX code or String The color for the global comment texts. Can be changed after initializing Barrage. Make sure your color is valid, otherwise no effect. For the initial Barrage structure, set structure.comment.color for your Barrage.
hover true Boolean The hover effect for the global comment texts & nodes. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.comment.hover for your Barrage.
poolNum 5 Int The maximum number of comments in one row of Barrage. Can be changed after initializing Barrage. Minimum: 1, Maximum: 7. For the initial Barrage structure, set structure.poolNum for your Barrage.
channelNum maxmimum rows / columns of comments in Barrage Int The maximum number of rows of your Barrage. Can be changed after initializing Barrage. Minimum: 5, Maximum: maxmimum rows / columns of comments in Barrage. For the initial Barrage structure, set structure.channelNum for your Barrage.
sentDelay 2 Int A factor you want to delay your comment to be sent out. Can be changed after initializing Barrage. Minimum: 2, Maximum: 20. For the initial Barrage structure, set structure.sentDelay for your Barrage.
recur true Boolean Whether you want your comment to be sent repeatedly, the barrage only sent a comment once if "recurStatus" is "false". Can be changed after initializing Barrage. For the initial Barrage structure, set structure.recurrence.recur for your Barrage.
recurDelay 7 Int Number of seconds you want to your comment to complete the transition on your display region. Can be changed after initializing Barrage. Minimum: 1, Maximum: 10. For the initial Barrage structure, set structure.recurrence.delay for your Barrage.
animation 'linear' String Animation for global comment texts & nodes. Current available animations: 'pop-in', 'quadratic', 'ease-in-quart', 'ease-in-cubic', 'linear'. Make sure your animation is valid, otherwise will change to 'linear'. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.animation.ani for your Barrage.
direction 'right' String Moving Direction for global comment texts & nodes. Current available directions: 'top', 'bottom', 'left', 'right'. For example, if choosing 'right', then comments will move from right to left. Make sure your direction is valid, otherwise will change to 'right'. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.animation.dir for your Barrage.
rand false Boolean Randoms start position for global comment texts & nodes. Only for 'top', 'bottom' directions. Set fixed when initializing Barrage, need to restart Barrage whenever want to change it. For the initial Barrage structure, set structure.animation.random for your Barrage.
Methods in Barrage 
        
  // Change poolNum for Barrage, make sure poolNum type is Int. 
  // Minimum: 1, Maximum: 7.
  barrage.setPoolNum(poolNum)

  // Change channelNum for Barrage, make sure channelNum type is Int. 
  // Minimum: 5, Maximum: maxmimum rows / columns of comments in Barrage.
  barrage.setChannelNum(channelNum)

  // Change sentDelay for Barrage, make sure sentDelay type is Int. 
  // Minimum: 2, Maximum: 20.
  barrage.setSentDelay(sentDelay)

  // Change recur for Barrage, make sure recur type is Boolean.
  barrage.setRecur(recur)

  // Change recurDelay for Barrage, make sure recurDelay type is Int. 
  // Minimum: 1, Maximum: 10.
  barrage.setRecurDelay(recurDelay)

  // Change commentSize and/or commentColor for Barrage. 
  // Make sure commentSize type is Int, commentColor is valid. 
  // Minimum for commentSize: 10.
  barrage.setComment(commentSize, commentColor)

  // Add comment text to Barrage, make sure commentText type is String.
  barrage.addCommentText(commentText)

  // Add all comment texts in commentTexts array to Barrage. 
  // Make sure comment in commentTexts array have type String.
  barrage.addAllCommentText(commentTexts)

  // Add comment node to Barrage, make sure commentNode type is DOM Object.
  barrage.addCommentNode(commentNode)

  // Add all comment nodes in commentNodes array to Barrage. 
  // Make sure comment node in commentNodes array have type DOM Object.
  barrage.addAllCommentNode(commentNodes)

  // Terminate Barrage, your cannot activate Barrage again after terminate it.
  barrage.terminateBarrage()
