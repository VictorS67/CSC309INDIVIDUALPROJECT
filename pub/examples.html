Barrage.js --A User-friendly Tool for Designing WebApps

Hi, this is the library for creating "Barrage".

Examples Basic barrage animation
Basic barrage animation with comment texts. 
        
  var displayRegion = document.querySelector('.example__1');
  displayRegion.style.height = '200px';
  displayRegion.style.width = '800px';

  var myBarrageStructure = {
    poolNum: 5,
    channelNum: 10,
    sentDelay: 2,
    recurrence: {
      recur: true,
      delay: 7
    },
    comment: {
      height: 30,
      width: 100,
      size: 18,
      color: 'black',
      hover: false
    },
    animation: {
      ani: null,
      dir: null,
      random: false
    }
  }
  
  var barrage = new Barrage(displayRegion, myBarrageStructure)
  barrage.addCommentText('CSC309 is very fun!');
  barrage.addCommentText('CSC309 is not fun!');
  barrage.addCommentText('CSC309 is interesting!');
  barrage.addCommentText('CSC309 is awful!');

  com_node.style['background-color'] = 'cornflowerblue';

Complex barrage animation
Complex barrage animation with different directions and animations. Animation and direction is setted before initializing Barrage, these two properties cannot be change afterwards. You can use terminateBarrage() to terminate the current barrage, and change animation and direction to restart. Note: Set the height and width of comment properly while changing the animation and direction.
